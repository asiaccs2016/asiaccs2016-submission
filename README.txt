DOP-Obfuscator is a dynamic opaque predicate obfuscator. It is able to 
insert dynamic opaque predicates (DOP) into different program
structures, including straight line code, branch and loop. 

DOP-Obfuscator is developed based on LLVM-Obfuscator, which is
released under UIUC open source licence. 

https://github.com/obfuscator-llvm/obfuscator
